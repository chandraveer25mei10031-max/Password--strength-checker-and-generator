# Project Statement
This project is a Password Strength Checker and Generator.
It checks password complexity and generates secure passwords for users.
Many users create weak passwords that compromise the security of their online accounts.
There is a need for a tool that evaluates password strength and provides helpful suggestions to ensure users can protect their accounts from unauthorized access 

# Scope of the Project
This project is designed for anyone managing online accounts who wants to create secure passwords. The tool is accessible to all users and offers real-time feedback to help improve password strength .

# High-Level Features
- Password strength analysis (categorizes as weak, medium, or strong) 
- Suggestions for improving password quality 
- Password generator to create strong, random passwords 
- User-friendly interface with instant feedback
